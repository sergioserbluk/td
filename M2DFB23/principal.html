Actividad integradora

El objetivo de esta actividad es familiarizarse con los conceptos CSS y etiquetas

Estilos en cascada. Elementos básicos de CSS3. Uso de selectores. Etiquetas semánticas de Html5.

Ejercicio 1: Uso de estilo interno

  1. Crear un archivo principal.html,
    1. Insertar un estilo interno para un párrafo existente. En el caso de no contener párrafo incorporar uno.
    2. Se emplea la etiqueta <style> de HTML y solamente se pueden incluir en la cabecera del documento (sólo dentro de la sección head>)
    3. El estilo interno consistirá en darle un color al párrafo.
    4. Comprobar en el navegador el cambio producido.
Ejemplo de párrafo

Somos una empresa que se dedica a la venta de relojes de mesa y pared con diseños únicos e innovadores. Utilizamos materia prima de calidad. Elaboramos productos a la medida del cliente.

Ejercicio 2: Hoja de estilo externa

  1. Crear un archivo CSS llamado miestilo.css
  2. En el archivo CSS escribir el siguiente estilo para la etiqueta <body>
    3. body { background-color: #eaf2e3; color: #333; font-family: Arial, sans-serif; margin: 0; padding: 0; }
  3. En la página HTML se enlaza el archivo CSS externo (miestilo.css), mediante la etiqueta <link>. Ver material teórico sobre CSS.
  4. Comprobar en el navegador el cambio producido.

Ejercicio 3: uso de clases

  1. En el archivo CSS creado, utiliza el selector de clase para crear una clase en la hoja de estilo de tal manera que el párrafo quede centrado, negrita y cursiva.
  2. Comprobar en el navegador el cambio producido.

Ejercicio 4: Uso de secciones.

En el archivo principal.html
  1. agregar una etiqueta <div> que incluya el párrafo creado anteriormente.
  2. agregar otro párrafo dentro de la misma división.
  3. dar estilo a toda la división empleando una clase.
  4. comprobar en el navegador el cambio producido
